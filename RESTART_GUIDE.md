# RESTART GUIDE - Karn Website

तुम्ही PC बंद करून चालू केल्यावर तुमची Website पुन्हा सुरू करण्यासाठी हे सोपे उपाय करा:

## १. एका क्लिकवर सुरू करा (Recommended)
`karn` फोल्डरमध्ये असलेल्या **`start_karn_website.bat`** या फाईलवर डबल-क्लिक करा.
- ही फाईल आपोआप दोन काळ्या खिडक्या (terminals) उघडेल.
- एकामध्ये Backend सुरू होईल आणि दुसऱ्यामध्ये Frontend.
- जेव्हा या दोन खिडक्या उघड्या असतील, तेव्हा तुमची website चालेल.

## २. मॅन्युअली सुरू करणे (जर वरील पद्धत नसेल चालली तर)
जर तुम्हाला स्वतः कमांड्स देऊन सुरू करायचे असेल तर:

### Step 1: Backend सुरू करा
1. `karn\backend` फोल्डरमध्ये Terminal उघडा.
2. `npm start` ही कमांड टाका.

### Step 2: Frontend सुरू करा
1. `karn\frontend` फोल्डरमध्ये Terminal उघडा.
2. `npm run dev` ही कमांड टाका.

## ३. Website उघडा
एकदा दोन्ही सर्व्हर सुरू झाले की, तुमच्या browser मध्ये खालील लिंक उघडा:
[http://127.0.0.1:5173](http://127.0.0.1:5173)

---

**महत्त्वाची टीप (Important Note)**:
जेव्हा तुम्ही VS Code किंवा Antigravity बंद करता, तेव्हा तुमची Website पण बंद होते.
पुन्हा सुरू करण्यासाठी तुम्हाला प्रत्येक वेळी **`start_karn_website.bat`** वर क्लिक करावे लागेल.

**Backend port**: [http://127.0.0.1:5000](http://127.0.0.1:5000)
